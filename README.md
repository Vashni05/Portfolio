PORTFOLIO
This project is an Online Portfolio Website developed using HTML, CSS, and JavaScript. 
The purpose of this website is to showcase personal details, skills, projects, resume, 
and contact information in a professional manner.

The website is responsive and accessible on different devices such as mobile, tablet, and desktop.

• To design a personal portfolio website
• To display skills and projects
• To create a responsive web design
• To implement a contact form
• To deploy the website online

• HTML – Structure of the website
• CSS – Styling and layout
• JavaScript – Interactivity (form handling)

The website is developed using HTML for structure, CSS for styling, and JavaScript 
for form validation. A responsive layout is implemented using CSS media queries.

The project files include:
• index.html
• style.css
• script.js
